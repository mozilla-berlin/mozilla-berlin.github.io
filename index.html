Events

We host Mozilla meetups every week, see the calendar below and make sure to sign up through the Mozilla Berlin Meetup Page.

Additionally we regularly host tech meetups, learning groups and other web-related gatherings, free of charge. Our space is sponsored and maintained by Mozilla, but we operate based on a trust model that enables volunteers to indepently host their own events.

Do you want to run an event in the Mozilla Community Space? Please get in touch here!
